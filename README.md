# Youtube-Detox
This chrome extension is designed to detox your brain from youtube. The extension should not require any login or anything like that and should just work out of the box when you navigate to youtube.
The following are settings for the extension that should be turned on by default.

## Features:
### Disable shorts
Never show any shorts on any youtube page ever. If a short is opened directly, then it is allowed to play, but no other shorts are loaded (cannot scroll up or down)

### Disable recommendations
Never show any (or delete from the page) videos that are not from a channel to which you are subscribed.

### Disable auto-play
Always make sure autoplay is turned off

### Disable side recommendations on videos
When watching a video, remove recommended other videos (whether subscribed or not) from the side

### Disable "people also search for"
Remove the "People also search for" when making specific searches

### Disable auto-preview
Stop the video from playing when hovering over the thumbnail.

### Add Friction
Add a configurable delay before opening the website shows. Configurable to appear either just when opening youtube, or when clicking a video, or on any youtube page, etc.
